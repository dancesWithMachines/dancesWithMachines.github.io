# Description
This is repository for my personal website.

All the pages are written in GitHub markdown language and then processed to html using "[Markdown to html github style](https://github.com/KrauseFx/markdown-to-html-github-style)" with some custom scripts.

I hope this approach will result in that minimalist / doc like look I was hoping for.

# Processing markdown
How to:
1. Download and setup [Markdown to html github style](https://github.com/KrauseFx/markdown-to-html-github-style)
2. Modify the path to convert.js (markdown-to-html) file in convert.py
3. Run convert.py
